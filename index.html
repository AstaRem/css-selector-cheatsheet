CSS selectors and specificity

Selector in CSS Name Description
tag name any HTML tag, it means type least specific
.class class name more specific than type
#id id name more specific than class
h1.special chaining selectors we use it, if there is .special class in (not outside!)h1 element
p > .class chaining selectors .class is direct child of p
main-list li descendant combinator li item is nested within < ul class="main-list" > . Therefore it is descendant of .main-list
h1, .menu multiple selectors we can apply the same rule, for example same fonts to multiple elements, using multiple selectors, separated by comma
!important not selector, but rule It will override any style, no matter how specific it is.
In p{color:blue !important} all p elements appear blue.
